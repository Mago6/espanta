Configuración de GitHub

¿Cómo obtener esta URL?

  1. Sube tu archivo "tareas.json" a GitHub
  2. Abre el archivo en GitHub y haz clic en el botón "Raw"
  3. Copia la URL de la barra de direcciones
  4. Pégala aquí y haz clic en "Guardar URL"

Ejemplo: https://raw.githubusercontent.com/tuusuario/turepo/main/tareas.json
